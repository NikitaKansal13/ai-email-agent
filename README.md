# AI Email Agent

An AI-powered email assistant built using Node.js, Gmail API, and Gemini API.

## Features
- Gmail OAuth Authentication
- Read unread emails
- Extract email content
- AI-generated summaries using Gemini
- Filter automated/security emails
- Modular architecture

## Tech Stack
- Node.js
- Gmail API
- Google Generative AI (Gemini)
- dotenv

## Future Work

- Draft generation
- Mark emails as read
